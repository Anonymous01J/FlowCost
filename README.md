# FlowCost

This is a [React Native](https://reactnative.dev/) app built with [Expo](https://expo.dev/) and [Tamagui](https://tamagui.dev/).

## Get Started

1. Install dependencies: `npm install`
2. Run the app: `npm start`

## Features

- **Cross-platform:** Runs on iOS, Android, and web.
- **UI:** Styled with the Tamagui component library.
- **Navigation:** Uses Expo Router for navigation.
